## Included assets

- `assets/profile.jpg` - color profile photo
- `assets/cover.jpg` - black and white hero cover photo
- `assets/Azlan_Resume_2026.pdf` - downloadable resume
- `assets/projects/` - project preview artwork

## How to use

1. Open `index.html` in a browser to preview locally.
2. Replace the placeholder GitHub and LinkedIn links in `index.html`.
3. Upload the folder contents to a GitHub repository.
4. Enable GitHub Pages in repository settings.

## Suggested repository name

- `azlan-portfolio`
- or `yourusername.github.io`

## Publish with GitHub Pages

1. Push all files to GitHub.
2. Go to **Settings** > **Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select branch `main` and folder `/root`.
5. Save and wait for GitHub Pages to publish the site.
