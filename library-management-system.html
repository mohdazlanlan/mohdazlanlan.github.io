Diploma Internship Project

Library Management System

A Diploma Internship Project developed using NetBeans, JavaScript, and SQL database to support library record management and daily library operations.

NetBeans JavaScript SQL database Internship

Project Summary

This Library Management System was developed during my Diploma internship as a practical software project. The system was designed to help manage library operations more efficiently by organizing book records, user/member information, borrowing details, and return tracking through a structured SQL database.

The project strengthened my foundation in software development, database design, and user-focused system building. It also gave me early hands-on experience in translating real workplace needs into a working application using NetBeans, JavaScript, and SQL database.

Project Focus

Library record management, borrowing/return workflows, database storage, and practical system development during internship.

Tools / Platform

NetBeans, JavaScript, SQL database, Internship
