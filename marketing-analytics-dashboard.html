UTM Power BI Course Project

Marketing Analytics Dashboard

Developed as a Power BI course project at Universiti Teknologi Malaysia (UTM), this sales and marketing dashboard summarizes e-commerce performance through interactive KPI cards, sales trends, product analysis, regional contribution, customer ratings, and key influencer insights.

Power BI Sales Analytics E-commerce Data Lazada & Shopee Customer Feedback

Project Summary

This project was developed for a Power BI course at Universiti Teknologi Malaysia (UTM). It presents a marketing analytics dashboard that consolidates e-commerce sales data into one interactive reporting page, helping users monitor total sales, total orders, average order value, customer performance, product performance, category contribution, and regional sales distribution.

The dashboard uses slicers for date, platform, region, and product so users can explore performance from different angles. In the dashboard snapshot shown, it highlights a total sales value of 31.02K, total orders of 1,994, and an average order value of 15.55, while also identifying "Farah" as the customer of the month. Overall, it serves as a practical decision-support dashboard for understanding market performance and improving product, regional, and customer-focused strategies.

Business Goal

Demonstrate practical Power BI skills in a UTM course project while turning e-commerce transaction data into clear KPIs and actionable visual insights.

Dashboard Focus

Sales performance, monthly trend analysis, product ranking, regional comparison, customer ratings, and key influencers of total sales.

Dashboard Analysis

Key Analytical Purpose

The dashboard is useful for identifying which products and regions generate stronger sales, understanding how platform and date filters affect performance, and interpreting customer rating patterns. It also includes a key influencers visual that helps explain what is most associated with higher total sales.

From a marketing perspective, this Power BI dashboard supports faster reporting and clearer decision-making. Users can spot strong-performing products, compare regional contribution, review monthly trends, and use customer rating insights to guide promotional planning and sales strategy.
