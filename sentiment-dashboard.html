Dashboard Project

Marrybrown Malaysia Customer Sentiment Dashboard

A real-time sentiment analysis system that transforms Foodpanda customer reviews into useful business insights for monitoring customer satisfaction, branch performance, complaint patterns, and management actions.

Python XLM-RoBERTa-base + EDA MongoDB Streamlit Foodpanda Reviews

Project Summary

Customer reviews on food delivery platforms are valuable for understanding brand reputation and customer satisfaction, but manually reviewing large volumes of unstructured feedback is slow and may delay business decisions. This project addresses that challenge by developing a real-time customer sentiment analysis system for Marrybrown Malaysia using Foodpanda reviews.

The system is designed for multilingual and code-switched Malaysian reviews, including Malay, English, and Manglish. It collects reviews through web scraping, cleans the text, classifies sentiment, stores prediction results in MongoDB, and visualizes the insights through a live Streamlit dashboard.

Business Goal

Help stakeholders monitor customer satisfaction, identify operational issues, and support data-driven decision-making.

Final Model

XLM-RoBERTa-base with Easy Data Augmentation was selected for stronger balanced performance using Macro F1-score.

Model Evaluation

Why XLM-RoBERTa + EDA?

Several approaches were tested, including SVM + TF-IDF, mBERT, and XLM-RoBERTa-base. Because the dataset was imbalanced across Positive, Neutral, and Negative classes, Random Undersampling, Class-Weighted Loss, and Easy Data Augmentation were also evaluated.

XLM-RoBERTa-base with EDA was selected because it delivered the strongest balanced performance based on Macro F1-score, making it more suitable for multilingual and imbalanced customer review classification.
